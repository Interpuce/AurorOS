# Working compiler toolchain

The bold entries are the compiler versions currently used to compile the official ISO files.

## List of C compilers that are confirmed to work:

- **gcc (GCC) 15.2.1**

You can check your gcc version using this command:

`gcc --version`

## List of C++ compilers that are confirmed to work:

- **g++ 15.2.1**

You can check your g++ version using this command:

`g++ --version`

## List of Assembly compilers that are confirmed to work:

- **NASM 3.01**

You can check your Netwide Assembler version using this 
command:

`nasm --version`
